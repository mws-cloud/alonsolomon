# הקמת צד השרת — פעם אחת בלבד

הפאנל ודף הנחיתה משתמשים ב-Supabase **הקיים** של האתר (פרויקט `emkrjhmlorbjqaltivpo`).
כדי להפעיל את עורך התוכן ואת רשימת הלידים בפאנל, נדרשים שלושה צעדים חד-פעמיים:

## 1. קבלת גישה לפרויקט ה-Supabase

מי שמחזיק בפרויקט (הלקוח או המתכנת הקודם) צריך להזמין אתכם:
**Supabase Dashboard ← Organization ← Team ← Invite member** (עם הרשאת Admin או Developer).

## 2. הרצת קובץ ההקמה

1. נכנסים ל-[Supabase Dashboard](https://supabase.com/dashboard) → הפרויקט של האתר
2. בתפריט הצד: **SQL Editor**
3. מדביקים את כל התוכן של `setup.sql` ולוחצים **Run**

הקובץ בטוח להרצה חוזרת ולא נוגע בנתונים קיימים. מה הוא עושה:
- יוצר את טבלת `landing_content` (הטקסטים של דף הנחיתה) עם הרשאות מתאימות
- מוסיף הרשאת קריאה/עדכון לידים למשתמשים מחוברים (לצורך הפאנל)

## 3. יצירת משתמש כניסה ללקוח

1. בתפריט הצד: **Authentication ← Users ← Add user ← Create new user**
2. מזינים את האימייל של עו"ד סולומון וסיסמה ראשונית
3. מסמנים **Auto Confirm User** ✓
4. מוסרים ללקוח את האימייל והסיסמה — איתם הוא נכנס לפאנל בכתובת:
   `https://a-solomon.com/program/admin/`

## בדיקה שהכל עובד

1. נכנסים לפאנל עם המשתמש שנוצר
2. טאב **לידים** — אמור להציג את רשימת הלידים (ריק אם עוד לא הגיעו)
3. טאב **תוכן הדף** — משנים טקסט כלשהו, לוחצים שמירה, ומרעננים את דף הנחיתה
   לוודא שהשינוי מופיע
